2. Узнать какие у них подключены репозитории. Необходимо узнать содержимое файла **sources.list**, для этого необходимо открыть его в редакторе **nano** командой: `sudo nano /etc/apt/sources.list`

   

3. Исправить ошибки в зависимостях установив необходимые пакеты, зависимости с которыми не удовлетворены.

   Для этого необходимо:

   1. После изучения их файла sources.list указать им что именно необходимо в нем исправить - у каждого индивидуально.

   2. Минимальный набор необходимых репозиториев, эталон, указан в каталоге с инструкцией, см текстовый файл

4. Выполнить последовательность команд:

   -  `sudo apt update`

   -  `sudo apt install *наименование пакета*` *(для каждого недостающего пакета индивидуально)*
