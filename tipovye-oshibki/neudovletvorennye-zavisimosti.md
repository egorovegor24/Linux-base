## Для Astra Linux SE 1.8.4

1. Узнать в заявке какие именно зависимости не удовлетворены.

    Это отображается или в терминале, если они устанавливают через терминал 

   Или во вкладке "Зависимости", если они устанавливают через GUI (графический интерфейс)

2. Узнать какие у них подключены репозитории

   Для этого необходимо узнать содержимое файла /etc/apt/sources.list, посмотреть они могут это следующей последовательностью команд: `sudo nano /etc/apt/sources.list`

3. Исправить ошибки в зависимостях установив необходимые пакеты, зависимости с которыми не удовлетворены Для этого необходимо: 

   1. После изучения их файла sources.list указать им что именно необходимо в нем исправить - у каждого индивидуально. Минимальный набор необходимых репозиториев, эталон, указан в каталоге с инструкцией, см текстовый файл

   2. Выполнить последовательность команд: 

      -  `sudo apt update`

      -  `sudo apt install *наименование пакета*` *(для каждого недостающего пакета индивидуально)*

4. Получить обратную связь
