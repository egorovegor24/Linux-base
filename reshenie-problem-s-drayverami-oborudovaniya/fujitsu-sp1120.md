## **Для Альт Образование:**

Для настройки выполните последовательно шаги:

1. Получите данные подключенного устройства Fujitsu для последующего создания файла с правилами, для этого выполните команду:

   `sudo lsusb`

   Для корректного отображения информации вам требуется понять, что\
   "Идентификатор производителя"(idVendor), "Идентификаторы продукта"(idProduct) информация о продукте будут написаны после текста ID \*\*\*\*:\*\*\*\*,\
   поэтому при создании правила просьба написать свои идентификаторы в соответствии с техническими характеристиками устройства.

2. Создайте файл правил в директории **/etc/udev/rules.d** для этого выполните команду:

    `sudo nano /etc/udev/rules.d/*наименование файла*.rules`\
   *Например:* **sudo nano /etc/udev/rules.d/99-fujitsu-sp-1120.rules**

3. Внесите данные в созданный вами файл используя текстовый редактор **nano**:\
   *Например для Fujitsu SP-1120:*\
   \# Fujitsu ScanSnap SP-1120\
   ATTRS\{idVendor}=="04c5", \
   ATTRS\{idProduct}=="1473", \
   MODE="0664", \
   GROUP="scanner", \
   ENV\{libsane_matched}="yes"

4. Выполните перезагрузку правил **udev**, для этого выполните последовательно команды:\
   `sudo udevadm control —reload`\
   `sudo udevadm trigger`

5. Переподключите устройство к компьютеру
